# nRF_Academy
nRF Study 워크스페이스 입니다.   
관련해서 스터디했던 프로젝트들을 업데이트하고 있습니다.

## Goal
- Zephyr RTOS 기반 펌웨어 개발 기술 습득
- BLE, WiFi 기술 습득

## Environment
![](./img/nrf7002dk.jpg)   
- **Board**: `nRF7002DK`
  - Host MCU: `nRF5340` (ARM Cortex-M33)
  - Co-processor: `nRF7002`
- **SDK**: `nRF Connect SDK 3.0.2`
- **OS**: `Zephyr RTOS`
